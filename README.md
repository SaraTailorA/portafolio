# Personal Portfolio & Pets Gallery

This project is a beginner-friendly web development exercise built with **HTML5**, **CSS3**, and **JavaScript**. It includes a personal professional portfolio and a separate pets gallery page, allowing me to practice semantic structure, navigation, responsive design, and basic interactivity.

---

## Project Overview

The main objective of this project was to apply the core concepts learned at **RIWI**, including:

* **Semantic HTML5:** Using tags like `<header>`, `<nav>`, `<section>`, `<article>`, `<form>`, and `<footer>`.
* **CSS3 Styling:** Applying colors, typography, spacing, borders, shadows, and responsive layouts.
* **JavaScript Basics:** Adding interactivity through buttons, dynamic text changes, alerts, and show/hide sections.
* **Navigation:** Implementing both internal anchors (`#`) and links between pages.
* **Responsive Design Basics:** Using flexible layouts, `grid`, `max-width`, and media queries for better display on different screen sizes.

---

## Project Structure

The repository contains the following files:

* `index.html`: The main portfolio page with personal information, work experience, projects, and a contact form.
* `mascotas.html`: A secondary page with a gallery showing my pets.
* `style.min.css`: The stylesheet that controls the visual design, layout, colors, typography, and responsive behavior.
* `script.js`: The JavaScript file that adds interactivity to the portfolio sections.

## Technical Features

* **Semantic structure** for better organization and readability

## How to Run

1. Clone this repository or download the project files.
2. Make sure all files are in the same folder:
   * `index.html`
   * `mascotas.html`
   * `style.min.css`
   * `script.js`
3. Open `index.html` in any modern web browser such as:
   * Chrome
   * Firefox
   * Edge
